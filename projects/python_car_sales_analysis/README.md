# Python Data Analysis Project: Car Sales Database

This project analyzes `projects/car_sales_analysis/Car_sales_row_data.csv` using Python (standard library only).

## What this project does
- Cleans and parses numeric fields with missing values.
- Measures data quality (missing values by numeric column).
- Finds top manufacturers by total sales volume.
- Computes average price by vehicle type.
- Calculates correlation metrics:
  - Horsepower vs price
  - Sales vs price

## Files
- `analysis.py` — main analysis script.
- `insights.md` — generated findings and business takeaways.
- `top_manufacturers.csv` — generated ranked output table.

## How to run
```bash
python projects/python_car_sales_analysis/analysis.py
```
